## Task Management (Cardboard Plugin)

### Task Format

```markdown
- [ ] Task description #tag1 #tag2 @due(YYYY-MM-DD)
  
  Optional context notes
  Links to [[related-notes]]
  
  - [ ] Subtask 1
  - [ ] Subtask 2
```

### Supported Formats

**Due dates**Â (all three formats work):

- `@due(YYYY-MM-DD)`Â - Cardboard format (preferred)
- `[due:: YYYY-MM-DD]`Â - Dataview format
- `ğŸ“… YYYY-MM-DD`Â - Tasks plugin format

**Completion timestamps**:

- `@completed(YYYY-MM-DDTHH:MM:SS)`Â - Cardboard format

**Auto-complete parent**:

```markdown
- [ ] Parent task @autocomplete(true)
  - [ ] Subtask 1
  - [ ] Subtask 2
```

Parent auto-completes when all subtasks done.

**Disable auto-date on daily note**:

```markdown
- [ ] Task on daily note @due(none)
```
